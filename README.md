# Система бенчмаркинга алгоритмов поиска ближайшего соседа

Эта программа предназначена для сравнения различных алгоритмов поиска ближайшего соседа, оценки их производительности и точности на различных распределениях данных.

## Возможности

- Сравнение скорости и объема памяти для разных алгоритмов
- Тестирование на различных распределениях входных данных
- Оценка полноты (recall) результатов поиска
- Поддержка различных метрик расстояния

## Реализованные компоненты

### Метрики расстояния
- Евклидова метрика

### Алгоритмы
- Наивный перебор (Brute Force)

### Распределения данных
- Равномерное распределение (Uniform)

## Требования

- C++20
- CMake 3.16 или выше
- Компилятор clang++

## Сборка

```bash
mkdir build
cd build
cmake ..
ninja
```

## Использование

### Запуск бенчмарка

```bash
./nn_search [размер_набора_данных] [количество_запросов] [размерность]
```

По умолчанию:
- размер_набора_данных = 10000
- количество_запросов = 100
- размерность = 10

### Запуск тестов

```bash
./nn_tests
```

## Расширение функциональности

Программа разработана с соблюдением принципов SOLID и легко расширяема:

1. Для добавления новой метрики расстояния:
   - Реализуйте интерфейс `DistanceMetric`

2. Для добавления нового алгоритма:
   - Реализуйте интерфейс `NearestNeighborAlgorithm`

3. Для добавления нового распределения данных:
   - Реализуйте интерфейс `DataGenerator`

## Формат результатов

Результаты бенчмарка сохраняются в файл CSV с информацией о:
- Названии алгоритма и метрики
- Распределении данных
- Размере набора данных и размерности
- Времени предобработки
- Использовании памяти
- Среднем/минимальном/максимальном времени запроса
- Полноте (recall) полученных результатов
